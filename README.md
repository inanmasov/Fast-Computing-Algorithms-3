Программа реализует арифметические операции в группе точек эллиптической кривой.Она позволять задавать параметры эллиптической кривой; задавать точку аффинными координатами и проверять ее принадлежность кривой; генерировать случайную точку эллиптической кривой (случайно генерируется x-координата точки и ищется квадратный корень в поле для определения y); выполнять сложение двух произвольных указанных пользователем точек либо точек, сохраненных в памяти программы. Удвоение точки проводится в якобиановых координатах, сложение в смешанных якобианово-афинных.
